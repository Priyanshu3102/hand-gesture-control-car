
# 🚗 Hand Gesture Control Car

## 📌 Overview

This project controls a robotic car using hand gestures detected via a camera.

## ⚙️ Technologies Used

* Arduino / ESP8266
* Python (OpenCV, MediaPipe)
* L298N Motor Driver

## 🔧 Features

* Real-time gesture detection
* Wireless control
* Low-cost implementation
